# SHOPPING_CART

**Project Overview:**

The Shopping cart project is a beginner friendly project that simulates a simple shopping cart system. Users can add items to the cart, removes items, view cart content. This project helps in understanding Python basics such as lists, loops, conditionals and functions.

**TECHNOLOGIES USED:**
*  lists
*  functions
*  loops
*  conditional statements


**FEATURES:**

* Add items to the cart
* Remove items to the cart
* View all items in the cart
* Exit the application

  **HOW TO RUN?**

  * Install Python(version 3.x.x)
  * Clone or download the repository
  * Open command or terminal prompt
  * navigate to the project folder
  * Run the program using:
     python project_name.py


   ** SAMPLE OUTPUT MENU:**

1. Add item
2. Remove item
3. View cart
4. Exit
